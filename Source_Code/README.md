Instructions for setting up the CW_Remote.ini file will be found in CW_Remote_Ini_Create.py. The CW_Remote.ini file must reside in the same folder as the CW_Remote.py file.

This runs in PyCharm on OS-X and in PyDroid3 on an Android tablet. So far, every method to package this script has failed, PyInstaller, Buildozer in Kivy VM ...

### Dependencies: ###
```
urllib3
botocore
boto3
kivy
```
